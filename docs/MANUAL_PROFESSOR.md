# 📘 Manual do Professor — GramQuest

Bem-vindo ao **GramQuest**! Este sistema foi feito para que você possa criar desafios divertidos de gramática para seus alunos.

## 1. Como Acessar o Editor

1. Abra o GramQuest no seu navegador através do link fornecido pela escola (ou jogando "Offline" se o app estiver instalado).
2. Na tela inicial, clique em **"🏠 Menu"** e adicione `/editor` na barra de endereços (exemplo: `gramquest.vercel.app/#/editor`).
3. Uma tela de **Login do Professor** aparecerá. Digite a senha (o padrão é `prof2026` a menos que tenha sido alterada pela direção).

## 2. Criando um Novo Módulo

Um "módulo" é como se fosse uma "fase" ou "listinha de exercícios". Exemplo: *Ortografia Geral*, *Verbos - Tempo e Modo*, etc.

- Na tela do Editor, clique em **"Criar Novo Módulo"**.
- Preencha o **Título** (exemplo: "Crase 101") e o seu **Nome/Autor** (exemplo: "Prof. Marcos").
- Agora, comece a adicionar Desafios clicando no botão flutuante vermelho **"+"** (Adicionar Desafio).

## 3. Tipos de Desafio

Você pode criar 4 tipos diferentes de perguntas para os alunos:

### 🎯 Múltipla Escolha
Como numa prova padrão. O aluno escolhe uma entre quatro alternativas.
- **Pergunta**: Digite o enunciado. Exemplo: "Qual destas frases possui erro de crase?"
- **Alternativas**: Marque a correta usando a "bolinha" e insira uma explicação/feedback opcional que o aluno verá se errar.

### 📝 Verdadeiro ou Falso
O aluno decide se a afirmação é Verdadeira ou Falsa.
- Útil para afirmações rápidas.
- Você pode incluir uma "Explicação" que aparecerá apenas se ele errar a resposta, ajudando na correção do raciocínio.

### 🧩 Preenchimento (Arrastar e Soltar)
O aluno verá uma frase com uma ___LACUNA___ e várias palavras aparecerão na "mochila" dele para ele arrastar até a lacuna.
- **Pergunta**: O menino fez a *<lugar onde a palavra vai entrar>*.
- **Palavras (Loot)**: Insira a palavra correta e adicione palavras erradas (distratores) na mochila.

### 📊 Ordenação de Frase
O aluno deve reconstruir uma frase arrastando os pedaços na ordem certa.
- **Fragmentos**: Ex: `A professora`, `explicou`, `bem`, `a matéria`.
- O jogo embaralha automaticamente e o aluno precisa organizar perfeitamente.

## 4. Salvando e Compartilhando

Ao terminar de adicionar todas as questões, role a página até o final e clique em **"Salvar Módulo"**. 
Pronto! Em poucos segundos, esse módulo já aparecerá na tela inicial de todos os alunos que estiverem com o jogo aberto. 

Quando a internet estiver fora do ar, você não poderá salvar novos módulos, mas os alunos ainda poderão jogar os módulos criados anteriormente (se tiverem sido salvos no dispositivo no Cache PWA).
